# fix/UI Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Fix 10 UX/RAG issues — persist OCR dates, clean document citations, serve original files, patient picker, fuzzy patient naming, docs table, ingestion stepper, resizable divider.

**Architecture:** FastAPI + LangGraph backend (Postgres/pgvector on Supabase), vanilla-TS + Tailwind frontend (`medagentic-dashboard`). Backend logic changes are TDD. Frontend verified with `tsc` + manual. All DOM-string interpolation keeps the existing `esc()` XSS guard.

**Tech Stack:** Python 3.11, SQLAlchemy, Alembic, LangGraph, pytest; TypeScript, Vite, Tailwind CDN.

**Caveman note:** Commit messages, PR body, README, docs in caveman style (user override). Code normal.
